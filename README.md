## 🔬 상세 소개  
&nbsp;코딩온 교육기관을 통해 진행한 첫 번째 프론트엔드 프로젝트입니다.  
<br />
&nbsp;알고 보면 더 재미있는 예술, 하지만 예상 외로 폐쇄적이고 이론을 배우기 녹록치 않은 분야라고 생각이 되었습니다.  
<br/>
&nbsp;이를 해결하기 위해 각 시대에 해당되는 미술품을 보여주고, 미술 관련 전시 및 뉴스를 보여주어   
예술에 관심이 있고 좋아하는 사람들의 편의를 위해 만들었습니다.
